# React + Vite

A React-based dashboard for managing and monitoring cameras.
This project supports pagination, search, status management (Active/ Inactive/ Maintanance), and toast notifications for actions.

Features : -
    Camera listing dashboard
    Pagination support for scalability
    Search/filter functionality
    Status management:
    - Active
    - Inactive
    Toast notifications for user actions
    Custom React hooks for data fetching
    Modular and scalable folder structure

Tech Stack used :-
    React
    JavaScript (ES6+)
    Axios for(API calls)
    CSS
